## Unity_Aufgabe3_RebeccaRawers_RPG

The unity file contains assets from:
https://pixel-boy.itch.io/ninja-adventure-asset-pack

Game controls:
Esc: Opens pause menu + interactable inventory and settings
Button E: Is to collect
Button F: Is to interact with NPC's

The game also uses a drag and drop system.
If the player hovers their mouse above the inventory slots that
carries an Item, he can use it by pressing the interaction button f
